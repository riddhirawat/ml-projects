# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Problem

Predict whether a patient has heart disease based on medical features.

## 📊 Dataset

* Source: Kaggle Heart Disease Dataset
* Features: age, sex, cp, chol, etc.

## ⚙️ Steps Performed

* Data Cleaning & EDA
* Feature Scaling using StandardScaler
* Train-Test Split
* Model Training:

  * Logistic Regression
  * Random Forest
  * XGBoost
* Hyperparameter Tuning (GridSearchCV)
* Model Evaluation:

  * Confusion Matrix
  * ROC-AUC Curve
* Model Explainability:

  * SHAP (Feature Importance + Waterfall Plot)

## 📈 Results

* Logistic Regression: ~81%
* Random Forest: 100%
* XGBoost: 100%

⚠️ Note: Perfect accuracy may indicate overfitting.
