# 📩 SMS Spam Detection using Machine Learning

## 📌 Overview

This project classifies SMS messages as Spam or Ham using NLP and Machine Learning techniques.

## ⚙️ Techniques Used

* Text Cleaning (NLTK, Regex)
* TF-IDF Vectorization
* Models:

  * Naive Bayes
  * Logistic Regression
  * SVM (optional)

## 📊 Results

* Logistic Regression Accuracy: ~96.6%
* Naive Bayes Accuracy: ~96.4%

## 🔍 Key Insights

* Spam words: free, win, prize
* Ham words: ok, go, come

## 🚀 Improvements

* Threshold tuning
* Character-level TF-IDF
* Better recall optimization
